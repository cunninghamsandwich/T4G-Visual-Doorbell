# Design Files

This folder contains the design files for the 3D printed components:

[stl-files](design-files/stl-files):  This contains .stl files for use directly in 3D printing software.

[step-files](design-files/step-files):  This contains .step files for use in CAD software.

These files were created using OnShape. You can view and edit the original design here: (<https://cad.onshape.com/documents/337ad5c4e913baa748541b1a/w/32a3bb0dc3194d3076de666c/e/9b7dc48c14887d3fe6e58874?renderMode=0&uiState=69dc90a44aa9f7ec8603dad2>)
